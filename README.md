Saint Xavier High School Spring Dance Site
A modern, responsive website created to showcase the annual Saint Xavier High School Spring Dance. This project highlights event details, announcements, and media in a clean, student‑friendly design. Built as a fun, creative coding project to bring the excitement of the dance to life online.

✨ Features
Responsive layout for phones, tablets, and desktops

Event schedule with clear timing and activities

Announcements feed for quick updates

Photo & video gallery to showcase dance highlights

Lightweight design for fast loading

🛠️ Tech Stack
HTML5

CSS3 (or Tailwind / Bootstrap if you choose)

JavaScript

⚠️ Disclaimer
This project is not officially endorsed, sponsored, or affiliated with Saint Xavier High School. It is a student‑created website made for learning, creativity, and showcasing the Spring Dance in an unofficial way.

🚀 Getting Started
Download or clone the project

Open index.html in your browser

Edit the text, colors, and media to match your vision

Deploy using GitHub Pages, Netlify, or Vercel

🎨 Customization
You can easily adjust colors, fonts, animations, and layout to match the Spring Dance theme. Add countdowns, transitions, or interactive elements to enhance the experience.
